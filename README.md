## Install:
```
brew install closure-compiler
npm ci
```

## e2e tests
```
npm run e2e
```
There are tests for simple page, fot iframe from the same origin, for iframe
from another origin.

```
npm run test
```

```
npm run eslint
```

```
npm run eslint-fix
```

```
npm run build
```
